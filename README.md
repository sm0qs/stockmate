<h3 align="center">
  Your all-in-one tool for managing and uploading stock photos.
</h3>

Stockmate is a desktop application designed to assist photographers in streamlining the process of uploading photos to multiple stock services and social media platforms. It focuses on efficient management of keywords, tags, and titles to optimize the submission workflow.

### Development Environment Note

**Important:** This project is primarily developed on **Windows**. I sometimes test on **Linux (KDE Plasma)**, but I cannot provide support for other platforms. I don't have the resources to maintain or troubleshoot macOS or other Linux environments.

Contributions that improve cross-platform support are welcome.

## [Security](SECURITY.md)

This project does not collect or transmit any user data. All operations are performed locally on the user's machine. However, as the application is still in its early stages, users should exercise caution and avoid using it with sensitive data until it reaches a more stable release.
Any security vulnerabilities discovered should be reported via GitHub Private Security Reporting.
